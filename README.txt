Chargify Mac OS X Dashboard Widget
----------------------------------

A Dashboard widget for Mac OS X which displays your Chargify site
stats.

Created by: Douglas Robertson
Version:    0.4  --  21 April 2011


Source Repository
-----------------

The latest files for this widget will always be found in the GitHub
repository at:

https://github.com/grasshopperlabs/chargify_mac_osx_dashboard_widget


Building
--------

To turn this collection of HTML, stylesheets, Javascript and images into
a working widget, simply copy all the project files into a new folder.
Rename the folder to 'Chargify.wdgt' and the folder icon will change
to the Dashboard icon. Double-click on the icon to install/run the
widget.
